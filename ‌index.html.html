Summary

The point of this project was to research the disparities that are going on with the Coronavirus. The use of Matplotlib and google Api was involved to gather the data needed to provide info for hospitals. Topics included Adults vs Elderly, All Bed Occupancy, All Bed vs ICU Bed, and ICU-Occupancy.
